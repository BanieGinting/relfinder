Release History

Note: An update to the latest RelFinder version usually just requires to exchange the SWF file.

--
Release - version 1.3 (2011/04/06):

  • Graphics and information adapted to Visual Data Web project
  • Config.xml adapted to the domain of the Visual Data Web project

--
Release - version 1.2 (2010/11/02):

  • Flex 3.5 compliant
  • Shortly delayed suggestion search (~0.75 sec.):
    • fewer SPARQL queries (reduced endpoint load)
    • faster suggestions on average (due to shorter query queue)
  • URI for SPARQL namespace added if no URI is given by the endpoint

--
Minor release - version 1.1.5 (2010/08/10):

  • The autocomplete language is no longer set to English by default. If no language code is provided, no autocomplete language is used.
    (This increases compatibility with SPARQL endpoints.)

--
Minor release - version 1.1 (2010/05/18):

  • Improved dataset configuration (more parameters):
    • disable/enable appending of '/sparql?'
    • switch HTTP method (POST and GET)
    • change autocomplete language filter

--
